WORKING CASE FOLDER
===================
Put your completed SAISIE PDFs here. The app will save the filled K138 PDF in this same folder.

Example:
- Input:  SAISIE À FAIRE_francompact 2025-dummy.pdf
- Output: K138_SAISIE À FAIRE_francompact 2025-dummy.pdf (or similar)

Select a PDF from this folder when processing.
